## ⚙️ How to Use PersonaPlex

Double-click the desktop icon to open the application. The program will open a dashboard showing your current personas. You can add a new persona by clicking the "Add New" button. Give your persona a name and choose a color for it. This helps you track which environment you currently use.

The application automatically isolates data for each persona. If you open a folder or a web link within a specific persona, the program keeps that information inside its own container. You can switch between personas using the menu at the top of the interface. 

You can remove a persona at any time. Right-click the name of the persona in the list and select "Delete". The program will ask for confirmation before it clears the associated data. This action removes the configuration files but leaves your primary system settings alone.

## 📁 Managing Your Data

PersonaPlex stores your data in a protected folder on your computer. You do not need to interact with these files directly. The software handles all reading and writing tasks to prevent errors. If you need to back up your profiles, use the "Export" button in the Settings menu. This creates a single file containing all your information. You can save this file to an external drive or cloud storage. Use the "Import" button to restore your personas if you move to a new computer.
